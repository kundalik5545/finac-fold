# Project rules

- use shadcn and tailwindcss
- use typescript with app router
- always create components and stitch them together
- make/add comments so peer review become easy
- create repetative components inside custome-components folder for example **color-picker.tsx**
- always return api response with status code which help debuging use lib/status-code.ts file status codes also add new if required.
- Always create responsive code use grid layout or flex as needed.
- always create schema inside lib/schema and types inside lib/types
- always add back button for pages
